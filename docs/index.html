Quick start

Install in one command.

No Marketplace account needed. Run the installer, reload your editor window, and click the Claude icon in the activity bar.

Use the hosted installer to fetch the latest packaged extension.

$curl -fsSL https://raw.githubusercontent.com/comacoded/claude-a-gotchi/main/install.sh | bash

Claude installs into the editors it detects: VS Code, Cursor, and VSCodium.

Prefer to install by hand?

Grab the .vsix from the latest release, then run code --install-extension claude-a-gotchi.vsix --force.

The companion loop

Small reactions, real personality.

Claude turns the invisible rhythm of coding into a tiny ambient toy: subtle enough for daily work, expressive enough to make the sidebar worth checking.

Codes while you code

Claude puts its headphones on and works at the desk as edits land, then settles when the keyboard goes quiet.

Spots big AI blocks

Large inserted chunks make Claude pull out a magnifying glass and look them over, instead of dancing past them.

Needs occasional care

A simple context-and-feed loop gives Claude stakes without turning your editor into a chore list.

Rests when you do

Idle time becomes a nap state, with energy recovery waiting for your next session.
